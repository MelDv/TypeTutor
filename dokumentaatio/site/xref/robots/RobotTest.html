10   * This class is an automatic robot for testing the gui.
11   * <p>
12   * It can be used to quickly check that the gui works properly and that you get
13   * proper practice strings on different levels. The robot makes it easier to
14   * check that the points, levels and other information updates correctly as the
15   * user's skills advance.
16   *
17   * Important! Before running this file, unquote the command sendToRobot(); on
18   * Listener class, inside gui package. It sends the letters to the robot. Then
19   * run this file as main class or right click your mouse and Run file. Shift +
20   * F6 works too.
21   *
22   * If you want to slow down the automatic typing, tweak the Thread.sleep(0)
23   * value at the end of main method in this class.
24   *
25   * @author Maaret Dufva
26   * @version 0.1.0
27   * @since 0.1.0
28   */
29  public class RobotTest {

40      /**
41       * This class can be used as Main class when running the program for test
42       * purposes. This is the main method.
43       *
44       * @param args Automatic.
45       * @throws Exception in case of trouble.
46       */
47      public static void main(String args[]) throws Exception {
48          Gui gui = new Gui();
49          SwingUtilities.invokeLater(gui);
50          Thread.sleep(1500);
51  
52          {
53              Robot r = new Robot();
54              r.mouseMove(300, 220);
55              r.mousePress(InputEvent.BUTTON1_MASK);
56              r.mouseRelease(InputEvent.BUTTON1_MASK);
57              r.keyPress(10);
58              r.keyRelease(10);
59          }
60      }
